# uav_slam_plan

This is the source code of my 2023 NIRS project.

The simulation is based on [PX4 SITL](https://github.com/PX4/PX4-Autopilot), ROS Noetic and Gazebo11. Using [ORB-SLAM2](https://github.com/raulmur/ORB_SLAM2) as visual SLAM system and [EGO-Planner](https://github.com/ZJU-FAST-Lab/ego-planner) for path planning.
